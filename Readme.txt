Project made by Joe Derrett as CSC8502 Coursework
Terraain created by me using the community version of Gaea2.

Code is a mix of my own and taken from the opengl Tutorials except otherwise stated.

For all attribution details see License.txt

To compile for yourself, you need my full Environment.gltf folder.
An incomplete version exists in this repo to show you where to put the contents of the full folder.
Get the Environment.gltf folder from: https://drive.google.com/file/d/1MzZexQ6TFTwhzLrY2guQDaI-6LKIP-uw/view?usp=sharing

Once you have the environment gltf, simply run the GraphicsTutorialsGLTF.sln file in Visual Studio.
